Design Decisions

  • Allocation toggle — Equal / Hours / Revenue / Custom, right at the top. Numbers update instantly.
  • Red row treatment — below-breakeven classes get a red background + "needs X more" hint so the owner immediately knows the gap
  • Overhead column — shows the actual allocated amount per class (varies by method), not just the flat $231
  • Sorted by margin % — best performers at top, problem classes at bottom
  • Clickable rows — navigate to class detail page for deeper analysis and what-if slider
